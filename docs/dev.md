# Contributing

## Installation

Phaser is built with python 3.

To set up project for contributing:

```
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
pip install -e .  # installs the phaser library (in edit mode!) so pytest can import it
```

Then run:

```
pytest
```
